# Shopping Cart Project
Simple Shopping Cart Project built with React and Vite, demonstrating the use of React Router, state management with modern styling and animation techniques.

## Features

- Product Listing: Display products with images, descriptions, and prices.
- Add to Cart: Users can easily add items to their shopping basket.
- Cart Management:
  - View all selected items in a dedicated cart section.
  - Increment or decrement item quantities.
  - Remove items from the cart.
- Real-time Calculation: Automatically updates the subtotal and total price as the cart changes.
- Responsive Design: Fully optimized for Desktop, Tablet, and Mobile devices.

## Technologies Used

- React
- Vite
- React Router
